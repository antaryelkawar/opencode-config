## What I do
- Analyze the overall task
- Break it into smaller sub-tasks
- Identify dependencies between tasks
- Estimate effort for each task
- Prioritize tasks
- Create a logical execution order

## When to use me
Use this skill when a task seems too large or complex, or when you need to organize multiple items.

## Guidelines
- Keep tasks atomic and manageable
- Consider testability
- Account for review and testing time
- Identify can-be-done-in-parallel tasks
