## What I do
- Generate API documentation
- Create README files
- Write usage guides
- Document configuration options
- Maintain changelogs
- Generate code comments and docstrings

## When to use me
Use this skill when creating new documentation, updating existing docs, or when you need to document new features.

## Guidelines
- Keep documentation clear and concise
- Include code examples where helpful
- Use consistent formatting
- Follow the project's documentation style
- Ensure docs are accessible to the target audience
