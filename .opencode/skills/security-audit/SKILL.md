## What I do
- Scan for common security vulnerabilities
- Check for input validation issues
- Identify authentication/authorization flaws
- Find data exposure risks
- Check dependency vulnerabilities
- Review configuration security
- Look for SQL injection, XSS, and other OWASP issues

## When to use me
Use this skill before deploying code, during code review, or when security is a concern.

## Guidelines
- Prioritize critical and high-severity issues
- Provide CVE references when available
- Suggest remediation steps
- Consider the attacker's perspective
